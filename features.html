Features

Explore the powerful capabilities of our oceanographic data platform designed for scientists, researchers, and policymakers to analyze ARGO float data with AI-powered insights.

Advanced Data Capabilities

FloatChat provides an end-to-end system for exploring ARGO oceanographic data using natural language queries and interactive visualizations.

NetCDF Data Ingestion

Process ARGO float NetCDF files and convert them to structured formats for analysis. Our system handles the complex data structures and metadata from oceanographic instruments.

DATA PROCESSING

Dual Database System

Utilize PostgreSQL for relational data storage and FAISS for efficient vector embeddings. This hybrid approach ensures both structured queries and semantic search capabilities.

DATA STORAGE

AI-Powered Queries

Leverage natural language processing through Groq API with a RAG pipeline. Ask complex questions about ocean data and receive insightful answers powered by AI.

AI ANALYSIS

Interactive Visualizations

Explore data through geospatial maps, depth-time plots, and profile comparisons. Visualize ocean parameters like temperature, salinity, and pressure across different dimensions.

Chat Interface

Ask questions about oceanographic data in natural language. Our AI understands context and provides detailed responses about ARGO float measurements and patterns.
